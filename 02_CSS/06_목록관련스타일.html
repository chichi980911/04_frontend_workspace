목록 관련 스타일(list ~)

list-style-type : 불릿기호를 변경시켜줄 떄 사용

list-style-image : 불릿기호로 이미지를 적용할 때 사용

list -style-position :불릿기호의 위치를 변경할 때 사용

        *순서없는 목록 (ul)
        선택자{
            list-style-type : disc|circle|square|none;
            list-style-image : url (적용시키고자 하는 이미지의 )
            list-style-position : inside | outside
        }
