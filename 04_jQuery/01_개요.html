*jQuery 구문 작성하기

jQuery 구문으로 dom요소를 다루는 기본 구문을 기본적으로 작성할꺼기 때문에
문서상에 요소들이 다 만들어지고 나서 실행되어야함

>순수 자바스크립트에서는 

        window.onload = function(){
          실행내용;
        }

        JavaScript 에서의 widow.onload 구문은 해당 문서의 요소들이 다 로드되고 뿐만아니라 연동되어 있는 외부 문서들도 다 가지고 온 뒤에 실행됨
        그리고 window.onload는 해당 이 문서에서 단 한번만 작성 가능(여러개 있다면, 마지막에 작성된 window.onload 함수 가 실행된다.)
      
      
      >jquery에서의 해당 내용은 3가지 구문으로 제공
      1) jquery(Document).ready(function(){
          실행내용
      });

      2)$doucumet.ready(fun(){
        실행내용;
      })

      3) $(fun){
        실행내용;
      }

      해당문서의 요소들만 로드되면 곧바로 실행된다.
