차이점 html() , text() - 태그 유무

Content 영역 (innerHTML , innerText) 관련메서드

* html(["문구"]) 메서드 : innerHTML 속성과 관련된 메서드

선택된 요소의 content 영역의 값을 리턴해주거나 또는 변경해주는 메서드
>$("선택자").html() : 선택된 요소의 content 영역 값을 반환해줌(html 태그까지 포함)
>$("선택자").thml("문구") : 선택된 요소의 content영역 값을 해당 문구로 변경해줌 (html태그가 존재할 경우 태그로써 해석해줌)

네이버로

* text(["문구"]) 메서드 : innerText 속성과 관련된 메서드

선택된 요소의 CONTENT영역의 값을 리턴해주거나 변경해주는 메서드
>$("선택자").text(); : 선택된 요소의 content영역 반환해줌(단, html 태그 포함x 오로지 텍스트만)
>$("선택자").(["문구"]) : 선택된 요소의 content 영역을 전달값으로 변경해줌 (단, html 태그가 존재할 경우 태그로 해석 x )
